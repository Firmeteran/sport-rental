## 🧰 Technologies Used

| Technology                                      | Description                                                                                                                   |

| [Go (programming language)](https://go.dev/)    | High-level, general-purpose programming language that is statically-typed and compiled.                                       |

| [Supabase](https://supabase.com/)               | Postgres development platform.                                                                                                |
| [GORM](https://gorm.io/index.html)              | The fantastic ORM library for Golang.                                                                                         |
| [GoDotEnv](https://github.com/joho/godotenv)    | Golang port of the Ruby [dotenv](https://github.com/bkeepers/dotenv) module project which loads env vars from a .env file.    |
| [Railway](https://railway.com/)                 | Full-stack cloud for deploying web apps, servers, databases, and more with automatic scaling, monitoring, and security.       |
